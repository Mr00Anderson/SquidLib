003/**
004 * Enum used for common targeting limitations (or lack thereof, in the case of AimLimit.FREE ). AimLimit.ORTHOGONAL will
005 * limit single targets or the centers/aimed-at-cells of AOE effects to cells directly, north, south, east or west of
006 * the user. AimLimit.DIAGONAL does the same but for northeast, southeast, southwest, or northwest. AimLimit.EIGHT_WAY
007 * limits the same things, but is less restrictive, allowing all cells AimLimit.ORTHOGONAL does as well as all cells
008 * AimLimit.DIAGONAL allows. AimLimit.FREE allows all cells within any range limit an ability may have.
009 * Created by Tommy Ettinger on 12/17/2015.
010 */
011public enum AimLimit {
012    FREE,
013    ORTHOGONAL,
014    DIAGONAL,
015    EIGHT_WAY
016}
